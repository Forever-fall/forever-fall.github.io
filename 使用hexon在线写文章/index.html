title: 使用hexon在线写文章
date: 2024-03-17
tags: hexo

首先安装hexon
git clone https://github.com/gethexon/hexon.git
安装依赖
npm install;npm run setup

提示

安装pnpm
npm install -g pnpm
执行
cd ./server && pnpm install -P && pnpm run setup

根据提示确认端口、hexo路径、登录用户名和密码

这里可以选择直接使用pm2管理hexon pnpm prd
